# MarcoVermeulen-ToDo_App

To-Do App

On the What's up, at the top of the screen you can change your name and it will keep it as is

You can then click on the create TODO and then pick one of the 2 category's

Click on the ADD TODO an it will appear in the color category you chose

You can then check a todo to say if it's completed or you can EDIT or DELETE each TODO

It is saved in local storage in the browser via JSON so data will stay there until deleted
